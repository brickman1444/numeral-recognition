How Does it Work?

This site uses a neural network to recognize the numerals. The neural network was trained on the MNIST data set of hand drawn numerals. The trained network was serialized into JSON. That JSON is instantiated at runtime by a Web Assmebly module and runs using the data from the HTML canvas. The result is piped back to the web page with Javascript.
